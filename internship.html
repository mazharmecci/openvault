Day 1: Setup & Fork the Repo

Today you'll set up your dev environment and fork the sample project.

📘 Step-by-Step Instructions
  1. Visit the sample GitHub repo.
  2. Click Fork (top-right) to copy it to your account.
  3. Open your terminal and run:
    git clone https://github.com/YOUR_USERNAME/fake-news-detection.git
  4. Open the folder in VS Code and install dependencies:
    npm install
  5. Run the project locally:
    npm run dev
